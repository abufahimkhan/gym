# USA Power Gym Svelte Template

## Installation

1. Extract the contents of this zip file.
2. Navigate to the `MainFiles/source` directory.
3. Run `npm install` or `pnpm install` to install dependencies.
4. To start the development server, run `npm run dev` or `pnpm dev`.
5. To build for production, run `npm run build` or `pnpm run build`.

## Production Build

- The production-ready files are in `MainFiles/build`.
- You can deploy these files to your preferred hosting provider.

## Customization

- All Svelte source code is in `MainFiles/source/src`.
- Update images, text, and styles as needed.
- For branding, edit `src/lib/constants.ts` and `src/components`.
