🎓 UNDERSTANDING THE NEW STRUCTURE
───────────────────────────────────────────────────────────────────

BEFORE (React):
  App.tsx → Component
  index.tsx → ReactDOM.createRoot
  pages defined manually with React Router

AFTER (SvelteKit):
  src/routes/+page.svelte → Page (automatic routing!)
  src/routes/+layout.svelte → Layout wrapper
  Routes = file structure (no config needed!)

🌍 ADDING NEW PAGES
───────────────────────────────────────────────────────────────────

To create an /about page:

1. Create folder:
   $ mkdir src/routes/about

2. Create file:
   $ touch src/routes/about/+page.svelte

3. Add content:
   <h1>About Us</h1>
   <p>Our story...</p>

4. Done! Visit: http://localhost:5173/about


💡 SVELTE 5 QUICK TIPS
───────────────────────────────────────────────────────────────────

Reactive Variables:
  let count = 0;  ← Automatic reactivity!

Event Handling:
  <button on:click={handleClick}>Click</button>

Loops:
  {#each items as item}
    <p>{item.name}</p>
  {/each}

Conditionals:
  {#if isReady}
    <p>Ready!</p>
  {/if}
