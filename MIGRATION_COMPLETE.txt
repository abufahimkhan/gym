🚀 QUICK START

1. Install dependencies:
   $ npm install
   or: $ pnpm install

2. Create environment file:
   $ cp .env.example .env.local

3. Add your Gemini API key to .env.local:
   VITE_GEMINI_API_KEY=your_key_here

4. Start dev server:
   $ npm run dev

5. Open: http://localhost:5173

🔄 ROUTING SYSTEM

SvelteKit uses FILE-BASED ROUTING:

File Location                      URL
───────────────────────────────────────
src/routes/+page.svelte           /
src/routes/about/+page.svelte     /about
src/routes/contact/+page.svelte   /contact
src/routes/blog/[slug]/+page.svelte /blog/:slug

To add a new page:
1. Create folder: src/routes/about/
2. Create file: src/routes/about/+page.svelte
3. Done! Visit /about

🌐 ENVIRONMENT VARIABLES

Create .env.local (don't commit this!):

VITE_GEMINI_API_KEY=your_actual_api_key_here

Notes:
- VITE_ prefix makes it accessible to browser code
- Without VITE_, variables are server-only
- Always restart dev server after changing .env.local

💡 SVELTE 5 BASICS

Reactive Variables:
  let count = 0;  ← Automatically reactive!

Event Handling:
  <button on:click={handleClick}>Click</button>

Two-Way Binding:
  <input bind:value={name} />

Conditional Rendering:
  {#if condition}
    <p>True</p>
  {:else}
    <p>False</p>
  {/if}

Loops:
  {#each items as item}
    <p>{item.name}</p>
  {/each}

🤖 AI FEATURE

Location: src/components/AiMotivation.svelte

How it works:
1. User enters fitness goal (e.g., "Heavy Deadlifts")
2. Component calls: getMotivationalQuote(goal)
3. Gemini API generates personalized quote
4. Quote displays with animation

Setup:
1. Add VITE_GEMINI_API_KEY to .env.local
2. Make sure API key has Generative AI access
3. Done! Feature works automatically

❓ TROUBLESHOOTING

Q: "npm: command not found"
A: Node.js not installed. Get it from nodejs.org

Q: "Cannot find module" errors
A: Run: npm install

Q: Port 5173 already in use?
A: Dev server will auto-use next available port, or:
   npm run dev -- --port 3000

Q: Environment variable not loading?
A: Check: 1) File is .env.local  2) Has VITE_ prefix  3) Restarted dev server

Q: AI feature returns "The only bad workout..."
A: Check your API key is correct and has Generative AI access
