🎓 KEY CONCEPTS TO UNDERSTAND

Routes (Important!):
  - Files in src/routes/ automatically become URLs
  - +layout.svelte wraps pages
  - +page.svelte is the page content

Components:
  - Live in src/components/
  - Reusable across routes
  - Have their own styling

Lib:
  - Utilities, constants, types
  - Import and use anywhere
  - constants.ts = your app data

Environment:
  - .env.local = secrets (don't commit!)
  - VITE_ prefix = client-visible
  - import.meta.env = access in code

⚠️ COMMON ISSUES & SOLUTIONS

Issue: Port already in use
Solution: npm run dev -- --port 3000

Issue: Cannot find module
Solution: npm install (and restart dev server)

Issue: API key not working
Solution: Check .env.local exists, has correct key, restart dev server

Issue: Tailwind styles not loading
Solution: Make sure index.html still has Tailwind CDN script tag

Issue: Components not showing
Solution: Check import paths start with ../lib/ or ../components/
